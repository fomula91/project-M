# Project M - 사쿠라 학원 비주얼 노벨

웹 기반 비주얼 노벨 프로토타입. Monogatari v2.0.2 엔진을 사용한 대화형 스토리 게임.

## 스토리

사쿠라 학원 봄 신학기 첫날, 전학생인 플레이어가 두 명의 히로인을 만나게 된다.

- **소라** - 조용하고 단정한 학급위원
- **하나** - 밝고 활발한 동급생

2개의 선택지를 통해 호감도가 변화하며, 최종적으로 3개의 엔딩으로 분기한다.

| 선택 | 효과 |
|------|------|
| 소라를 도와준다 | 소라 호감도 +2 |
| 하나를 따라간다 | 하나 호감도 +2 |
| 도서관 (소라와 함께) | 소라 호감도 +1 |
| 옥상 (하나와 함께) | 하나 호감도 +1 |

### 엔딩

- **소라 엔딩** - 소라 호감도 > 하나 호감도
- **하나 엔딩** - 하나 호감도 > 소라 호감도
- **우정 엔딩** - 호감도 동점

## 기술 스택

- **엔진**: [Monogatari](https://monogatari.io/) v2.0.2
- **실행 환경**: 로컬 HTTP 서버
- **에셋**: CSS 컬러 배경 (Phase 1, 이미지 미사용)

## 실행 방법

```bash
cd project-m
python3 -m http.server 8080
```

브라우저에서 `http://localhost:8080` 접속.

## 프로젝트 구조

```
project-m/
├── index.html          # 진입점
├── js/
│   ├── options.js      # 엔진 설정
│   ├── storage.js      # 게임 변수 (호감도, 플래그)
│   ├── script.js       # 캐릭터 정의 + 스토리 스크립트
│   └── main.js         # 초기화
├── engine/             # Monogatari v2.0.2 엔진
├── style/main.css      # 커스텀 스타일
└── assets/             # 에셋 디렉토리 (Phase 2용)
```

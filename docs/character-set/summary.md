## 소라 (Sora)

- **기본:** 재학생, 학급위원. 주인공과 같은 반.
- **성격 핵심:** 내성적(혼자가 편하다 믿음) / 조심스러움(먼저 다가가는 것을 두려워함) / 섬세한 정성(몰두하면 변모)
- **심리 구조:** 1학년 때 왕따 경험이 원점. **"혼자라면 상처받지 않는다"**는 생존 전략으로 벽을 세움. 외로움·불신·침묵의 3결핍.
- **구원 조건:** 구조나 해결이 아닌, **이해받는 것**과 **곁에 있어주는 것**. 억지로 바꾸려 하지 않고, 침묵을 견디며, 불편함을 지적하지 않는 사람.
- **서사 아크:** 닫힌 벽(D1) → 균열·종이학 선물(D2) → 감정 인지(D3) → 과거 직면·눈물(D4) → "이번에는 제가 먼저 말할게요" 고백(D5)
- **상징:** 종이학(마음의 표현), 석양(감정 고조), "봄은 금방 지나간다" 창틀 낙서(유우의 흔적)

> 상세: [`character-sora.md`](character-sora.md)

## 하나 (Hana)

- **기본:** 재학생, 유우가 이어준 소라의 친구. 분위기 메이커.
- **성격 핵심:** 밝음(가면 — "웃으면 미움받지 않는다") / 불안(이면 — 타인의 승인에 의존) / 용기(성장 — 가면을 벗는 결심)
- **심리 구조:** 중학교 때 왕따 경험 → **"밝은 척"**이라는 가면이 자기 보호 기제. 소라의 벽과 같은 상처의 정반대 전략. 소라를 돕지만 **"고치려 드는"** 방식이 되어 오히려 벽을 높임.
- **서투른 구원:** 과거 자신의 부정 + "나는 변했다"의 증명 + 인정 욕구가 선의와 뒤섞여, 소라에게는 강요로 전달됨.
- **서사 아크:** 밝음의 가면(D1) → 진지함의 시작(D2) → "굳이 웃지 않아도 될 것 같은 기분" 가면 인지(D3) → 과거 직면·첫 진짜 미소(D4) → 재고백·눈물 속 환한 웃음(D5)
- **상징:** 종이 왕관(장난과 애정), 옥상 벚꽃(밝음), 수영장 석양(취약함과 진실)

> 상세: [`character-hana.md`](character-hana.md)

## 하루 (Haru)

- **기본:** 전학생, 2학년 중간 편입. 주인공이자 플레이어 아바타.
- **성격 핵심:** 관찰자(먼저 지켜보고 상황을 읽음) / 공감형(상대의 감정을 자기 것처럼 느낌) / 성장형(관망에서 행동으로 변화)
- **심리 구조:** 전 학교에서 왕따·자살 충동 경험(게임 내 미언급) → 구원받은 기억이 타인의 아픔에 본능적으로 반응하는 근원. **구원의 순환** — 받은 것을 돌려주는 동기.
- **유우와의 차이:** 유우(비경험자의 선의, 현상유지) vs 하루(경험자의 공감, 적극 개입). 유우가 방향을 제시한 사람이라면, 하루는 그 방향으로 **함께 걸어가는 사람**.
- **서사 아크:** 관찰자 도착(D1) → 기대감·관여(D2) → 감정 인지(D3) → 유우의 조언으로 자기 감정 직면(D4) → "이번엔 솔직하게" 행동(D5)
- **상징:** 선택지(주인공만의 전유물, 방향→깊이→진정성 진화), "친구"라는 안전한 말(감정 회피→직면), 봄·전학(새로운 시작)

> 상세: [`character-haru.md`](character-haru.md)

## 유우 (Yuu)

- **기본:** 졸업생(작년 졸업). 소라와 하나를 이어준 선배, 미스터리 축.
- **성격 핵심:** 관찰자(기록하는 사람) / 연결자·허브(소라↔하나 유일한 다리) / 조언자(방향만 제시)
- **심리 구조:** 왕따 비경험자. 정의감과 관찰력으로 다가갔으나, 상처를 **체감**하지 못해 현상유지를 택함. **불완전한 구원자** — 균열은 냈으나 벽을 허물지 못함.
- **퇴장의 의미:** 졸업으로 허브가 빠지자 소라↔하나의 불완전성이 드러남. 그 빈자리가 하루의 서사 진입을 정당화.
- **서사 아크:** 흔적·실루엣(D1~3) → 등장·과거 공개·조언(D4) → 부재 속 존재·편지(D5)
- **상징:** "봄은 금방 지나간다" → "이 봄을 잊지 마" → "봄은 끝나지 않아" (유한→기억→영원), 실험 노트(기록하는 사랑)

> 상세: [`character-yuu.md`](character-yuu.md)

## 관계 구조도

```
소라 ←──── 거울 관계 ────→ 하나
(벽)    닮은 상처,             (가면)
        정반대 전략
  ↑                              ↑
  │ 벽에 균열                     │ "가면 벗어도 괜찮아"
  │                              │
  └──────── 유우 (허브) ─────────┘
            졸업 → 빈자리
                 ↓
              하루 (후계자)
         관찰 → 이해 → 행동
```

- **소라↔하나:** 같은 왕따 상처, 정반대 생존 전략. 닮았기에 이해할 수 있지만, 닮았기에 직면하기 두려운 **고슴도치 딜레마**.
- **유우:** 소라와 하나를 연결한 허브이자 완충재. 졸업으로 빈자리가 발생.
- **하루:** 유우의 후계자. 유우가 끝내 하지 못한 일 — 소라의 벽을 허물고, 하나의 가면을 벗기는 것 — 을 실행하는 인물.

## 테마 요약

> **핵심 테마:** "상처를 이해하는 것과, 상처를 고치려 드는 것은 다르다."

- **하나의 접근:** 상처를 **고치려** 든다 — 밝음을 강요하고, 변화를 요구하고, 자기 방식으로 구원하려 함. 진심이지만 소라에게 닿지 못함.
- **유우의 접근:** 상처를 **지켜본다** — 관찰하고 기록하고 기다림. 방향은 제시하되 직접 다루지 못함. 선의이지만 불완전.
- **하루의 접근:** 상처를 **이해하고 곁에 있어준다** — 같은 상처의 경험자이기에 체감으로 공명. 억지로 바꾸지 않으면서도 함께 움직임.

세 접근의 대비가 이 이야기의 감정적 골격이다.
